### Rust Language Enablement

Crucible now supports **Rust** as a first-class citizen! This release establishes the foundation for `rsfulmen`, the Rust helper library for the Fulmen ecosystem.

**What's Included:**

- **Scaffolding**: Complete `lang/rust` crate structure with Cargo workspace and MSRV 1.70 configuration.
- **Code Generation**: Automated bindings for:
  - **Foundry**: Exit codes (`foundry::ExitCode`)
  - **Fulpack**: Archive types (`fulpack::ArchiveFormat`, `fulpack::ArchiveEntry`)
  - **Fulencode**: Encoding types (`fulencode::EncodingFormat`)
  - **Fulhash**: Hashing types (`fulhash::Algorithm`)
- **Sync Pipeline**: `make sync` now hydrates `lang/rust/schemas`, `lang/rust/config`, and `lang/rust/docs` from the SSOT root.
- **Quality Gates**: Full integration with `make check-all` including `rustfmt`, `clippy` (pedantic), and `cargo test`.
- **Coding Standards**: New [Rust Coding Standards](docs/standards/coding/rust.md) and [Rust Testing Patterns](docs/standards/testing/language-testing-patterns.md#rust--claptokio-clis).

### Documentation

- Added `ADR-0013`: Rust Sync Pattern Validation.
- Updated `config/taxonomy/languages.yaml` to mark Rust as `active`.
