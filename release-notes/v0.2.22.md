### Goneat Tools Schema Enhancement

Crucible's goneat tools configuration schema now supports **Node.js** and **Python** package managers as installation methods!

**What's Included:**

- **Schema Update**: Extended the `kind` enum in `schemas/tooling/goneat-tools/v1.0.0/goneat-tools-config.schema.yaml` to include:
  - `"node"`: For tools installed via npm, pnpm, yarn, or other Node.js package managers
  - `"python"`: For tools installed via pip, uv, poetry, or other Python package managers
- **Sync Propagation**: Automatically synced to all language wrappers:
  - `lang/python/schemas/tooling/goneat-tools/v1.0.0/goneat-tools-config.schema.yaml`
  - `lang/typescript/schemas/tooling/goneat-tools/v1.0.0/goneat-tools-config.schema.yaml`
  - `lang/rust/schemas/tooling/goneat-tools/v1.0.0/goneat-tools-config.schema.yaml`

**Impact:**

This enables goneat (the Fulmen ecosystem's tool manager) to install formatting and development tools using the appropriate package managers for each ecosystem, improving tool discovery and installation reliability.
